# 💪 Calculadora IMC
O objetivo desse projeto é criar uma calculadora de Índice de Massa Corporal (IMC), trazendo como resultado para o usuário a classificação de acordo com os valores colocados no input.


## 🔨 FERRAMENTAS
- HTML5
- CSS3
- JavaScript
